# Project Overview

## Purpose

NSRC VMS replaces manual, paper-based volunteer management with a digital platform that handles the complete lifecycle of volunteer operations: from onboarding and event coordination to attendance verification, incident documentation, and performance reporting.

## Target Users

- **Administrators** — Full system control, user management, analytics, backups
- **Officers** — Event creation, attendance marking, incident review, announcements
- **Members** — Event joining, attendance viewing, incident reporting, profile management
- **Organizations** — Group-level event participation and member oversight
