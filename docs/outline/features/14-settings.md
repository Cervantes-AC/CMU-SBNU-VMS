# Settings & Preferences

Seven settings sections for comprehensive user control:

- **Theme** — Light (emerald), Dark (slate), NSRC Classic (crimson/gold) — 4-mode cycle
- **Notifications** — Configure notification preferences per category
- **Account Security** — Password change, MFA setup, session management
- **Data & Storage** — Cache management, offline data, export personal data
- **About** — App version, licenses, acknowledgments
- **User Guide** — In-app help documentation
- **Emergency Hotlines** — Quick-access emergency contact information
