# Import / Export

Data portability features supporting CSV and JSON formats for any collection.

- **CSV Import** — Row-by-row processing with validation and progress tracking
- **JSON Import** — Bulk document import with schema validation
- **CSV Export** — Collection export as CSV with configurable field selection
- **JSON Export** — Collection export as raw JSON
- **PDF Export** — Formatted PDF reports with headers, footers, tables, and summaries
- **Audit Logging** — All import/export operations are recorded in the audit trail
