# Audit Trail

An append-only, immutable audit trail records every significant system action. Audit logs cannot be updated or deleted, ensuring a tamper-proof history.

- **Actions Logged** — Auth events, CRUD operations, role changes, admin impersonation, SOS triggers
- **Severity Levels** — Info, Warning, Critical
- **Data Captured** — Action, category, user name, timestamp, old/new values, IP address
- **Viewing** — Officers and admins can filter and view audit logs
