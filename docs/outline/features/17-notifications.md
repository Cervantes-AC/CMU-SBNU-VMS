# Notification System

Multi-channel notification delivery combining Firebase Cloud Messaging, local notifications, and in-app alerts.

- **FCM Token Registration** — Automatic token management per device
- **Foreground Handling** — In-app notification display and routing
- **Background Handling** — System notification tray integration
- **Local Notifications** — Repeated display for critical alerts (SOS)
- **Notification Channels** — Android-specific channel configuration
- **Vibration Patterns** — Distinct patterns for normal vs. emergency notifications
- **Cloud Function Retry** — Automatic retry for failed notification deliveries
