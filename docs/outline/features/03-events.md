# Event Management

Events are the central organizing unit of the system. Officers and admins can create, edit, and manage volunteer events, while members can request to join.

## Event Lifecycle

- **Creation** — Title, date/time, venue, activity type, description, assigned volunteers
- **Statuses** — Upcoming → Ongoing → Completed (or Cancelled)
- **Join Requests** — Members submit join requests; officers/admins approve or deny
- **Assignment** — Officers can directly assign volunteers to events
- **Detail View** — Full event info, assigned volunteers list, join request queue, attendance records
