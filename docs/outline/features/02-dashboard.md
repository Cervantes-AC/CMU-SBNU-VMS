# Dashboard System

Four role-specific dashboards provide tailored views with relevant statistics, quick actions, and navigation to role-appropriate features.

- **Admin Dashboard** — System-wide analytics, user count, event stats, incident overview, recent activity
- **Officer Dashboard** — Assigned events, pending join requests, attendance summary, incident alerts
- **Member Dashboard** — Upcoming events, personal attendance, duty records, announcements
- **Organization Dashboard** — Group events, member attendance, organization-level stats

All dashboards feature glassmorphism design with stat cards, skeleton loaders, shimmer effects, and pull-to-refresh capabilities.
