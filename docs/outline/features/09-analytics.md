# Analytics & Reporting

Comprehensive analytics and reporting suite providing insights into volunteer operations.

## Analytics Dashboard

- Event statistics (total, upcoming, completed, cancelled)
- Incident trends and severity distribution
- Attendance rates and patterns
- Service hours accumulation over time
- fl_chart-powered visualizations (bar, line, pie charts)

## Report Types

- **User Activity Reports** — Individual volunteer activity summaries
- **Transaction Reports** — Event participation and attendance records
- **Audit Trail Reports** — System action history with severity filtering
- **System Usage Reports** — Overall platform usage statistics
- **Custom Reports** — Saveable report configurations for repeated generation
