# Audio Assets

This directory contains audio assets for the CMU NSRC Volunteer Management System.

## SOS Alarm

| File | Description |
|---|---|
| `sos_alarm.wav` | SOS emergency siren tone played in-app via `AudioService` when an SOS alert is received |

The file is auto-generated by the app build process. To replace with a custom alarm:
1. Replace `sos_alarm.wav` with your audio file (WAV or MP3)
2. Update `AssetSource` in `lib/data/services/audio_service.dart` to match the filename
3. Update `AndroidNotificationDetails` sound in `lib/data/services/notification_service.dart` if needed

Recommended specs:
- Format: WAV or MP3
- Duration: 3–10 seconds (looped in-app)
- Sample rate: 22050 Hz or higher

## Platform Setup

| Platform | Custom notification sound |
|---|---|
| Android | `android/app/src/main/res/raw/sos_alarm.wav` — referenced by `notification_service.dart` |
| iOS | `ios/Runner/sos_alarm.wav` — in-app playback via `audioplayers`; push notifications use default sound |

This directory is declared in `pubspec.yaml` under `flutter.assets`.

## In-App Usage

```dart
AudioService().playSosAlarm();   // plays in a loop
AudioService().stopSosAlarm();   // stops playback
```
